The general search allows you to search across all indexed fields.

To search in a specific field use fieldname:value. The following fields are available:

  • StudyDescription
  • SeriesDescription
  • PatientName
  • PatientID
  • AccessionNumber

You can combine multiple search parameters with the boolean operators AND and OR. Those needs to be capitalized.
